Windows Server 更新服务Windows Server Update Services

服务介绍

Windows Server 更新服务由微软官方提供，方便企业内部快速部署微软产品（包括 Windows 、 Office 、 SQL Server 、 Visual Studio 、 .NET Framework 等）更新及补丁的服务。

为了让大家能够快速、及时的下载安装微软更新及补丁，你只需要经过简单的设置，便可以使用由 We 提供的校园网高速 Windows Server 更新服务。

设置方法

  1. Windows 8 、 Windows 7 、 Windows Vista 、 Windows Server 2012 、 Windows Server 2008 R2 、 Windows Server 2008 的用户，可以通过在『开始菜单』中输入gpedit.msc；Windows XP 、 Windows Server 2003 的用户，在『开始菜单』中点击「运行...」，在弹出的窗口中输入gpedit.msc来启动『组策略编辑器』；
  2. 在『组策略编辑器』左侧窗格中，依次选择「计算机配置」、「管理模板」、「Windows 组件」、「Windows Update」，并在右侧窗格中双击选择「配置自动更新」；
  3. 在『配置自动更新』的属性窗口中，选择「启用」，并按照你自己喜欢的方式来设置一种更新方式，设置完成后单击「确定」按钮；
  4. 在『组策略编辑器』右侧空格中，继续双击选择「指定 Intranet Microsoft 更新服务器位置」；
  5. 在『指定 Intranet Microsoft 更新服务器位置』的属性窗口中，选择「启用」，并在两个输入框内均填入http://wsus.neusoft.edu.cn:8530，设置完成后单击「确定」按钮；
  6. 关闭『组策略编辑器』窗口。

恭喜你，你已经成功设置你的操作系统从 We 服务器来检测、下载并安装微软更新及补丁啦！

注意事项

  1. 由于 We 服务器下载速度比较快，你可能在短时间内下载大量更新及补丁，请注意安排补丁安装时间，以免在学校熄灯后，由于电池续航时间问题而导致更新及补丁安装到一半被中断；
  2. 虽然是在 We 服务器进行更新检测与安装，但是依然使用的是微软 Windows 操作系统的更新机制，请确定你所使用的操作系统是微软原版，并且不要与第三方工具（如 360 安全卫士、 QQ 安全管家等）同时对你的系统进行更新操作，否则可能会由于多套不同的更新机制而产生冲突；
  3. 当你由于放假、毕业等原因离开校园网，或者由于其他原因不想再继续从 We 更新你的系统，你可以参照上面的设置，在第2步和第4步操作中，选择「未配置」来停止使用由 We 提供的 Windows Server 更新服务。
