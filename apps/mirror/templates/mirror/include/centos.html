CentOS The Community ENTerprise Operation System

对于 CentOS 5.x, 6.x, 7.x 版本系统，均可运行下面命令修改系统初始配置文件： 

sed -e "s/^mirrorlist/#mirrorlist/g" -e"s/#baseurl/baseurl/g" -e "s/mirror\.centos\.org/mirrors.neusoft.edu.cn/g" -i /etc/yum.repos.d/CentOS-Base.repo

对于 CentOS 6.x 版本 x86_64 平台系统，还可运行下面命令配置 SCL 仓库： 

yum -y install centos-release-SCL
sed -e "s/mirror\.centos\.org/mirrors.neusoft.edu.cn/g" -i /etc/yum.repos.d/CentOS-SCL.repo
