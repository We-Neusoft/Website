常见问题

  1. 哪些播放器可以播放？

    目前确定可以正常播放的只有 Windows Media Player 。

  2. 为什么点击在线播放按钮后没有播放？

    出现这种问题，是因为你的浏览器没有将 mms 协议关联到 Windows Media Player 所导致，你可以尝试重新关联 mms 协议即可。或者，你可以右键点击按钮复制链接地址，然后手动打开 Windows Media Player ，再按键盘组合键ctrl+U粘贴该地址即可播放。

  3. 为什么有时播放节目会卡顿或无法播放？

    由于 We 的硬件能力原因，以及节目提供源东北大学及大连理工大学接收到的节目信号有时不稳定，因此转播过来的节目可能会出现播放异常现象。只要在日后有足够的能力，使用硬件自行采集高清信号后，便可大大减少此类现象。对此引起的不便，敬请谅解。

  4. 喜欢看一档节目，可以帮助录制吗？

    可以的，请尽早与 We 联系，以方便安排录制计划。在不与其他录制计划冲突并且信号稳定的前提下，我们将尽力完整录制。
